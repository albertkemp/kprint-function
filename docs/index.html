kprint()

About

KEMP is the programming language made in python, developed by Albert Kemp. The print function name is kprint(), short for KempPrint() which allows you to print strings.

Syntax

#kprint code
command = input()
if command.startswith("kprint('") and command.endswith("')") or command.startswith('kprint("') and command.endswith('")'):
string = command[8:-2]
print(string)
else:
print('ERR:command not found')

Updated versions allow you to use KempPrint() instead of kprint() and print numbers.

An updated version allowing KempPrint() as well as kprint()

command = input()

command = command.strip()
# Use parentheses to group the logical operators
if (command.startswith("kprint('") or command.startswith("KempPrint('")) and command.endswith("')") or (command.startswith('kprint("') or command.startswith('KempPrint("')) and command.endswith('")'):
# Check which type of quotation mark is used in the command
if command.endswith("')"):
# Split the command by single quotation marks and get the second element, which is the message
string = command.split("'")[1]
else:
# Split the command by double quotation marks and get the second element, which is the message
string = command.split('"')[1]
